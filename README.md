# ✈ Flight Operations & Delay Intelligence Dashboard

## 📌 Project Overview

Flight Operations & Delay Intelligence Dashboard is an end-to-end Data Analytics project developed using PostgreSQL and Power BI to analyze airline operations, airport performance, flight delays, and weather-related risks.

The project processes more than **50,000 flight records** and provides actionable insights through interactive dashboards, KPI monitoring, SQL analysis, and data visualization.

## 📈 Key Insights

* Hawaiian Airlines recorded the highest on-time performance.
* Spirit Airlines recorded the highest average delay.
* ATL was identified as the busiest airport.
* Blizzard conditions produced the highest weather delay risk.
* Carrier Delay was the most common delay category.
